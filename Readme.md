# Armut

Armut is a service provider application.

## Installation

Armut is available through CocoaPods. To install it, simply add the following line to your Podfile:

```bash
pod 'Kinfgfisher'
pod 'SwiftyJSON'
pod 'Alamofire'
```

Finally

```bash
pod install
```
